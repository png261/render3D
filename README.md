## How to run
Download `render3D.jar` at [releases page](https://github.com/png261/render3D/releases) 

Then run with command
```sh
java -jar render3D.jar
```
## How to use

Drag mouse or use arrow keys to rotate

Space + drag mouse to move

Scroll to roateY

Ctrl + scroll to zoom
